A plugin that renders an ICS calendar.

== Description ==

This plugin renders a remote ICal calendar to your webpage, you can set the colors for categories.
This plugin supports to import a secondary ICS-based calendar, to display holidays and vacation days in background.

You also can create a PDF output of the annual calendar using holidays and vacation days in background.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/kronos` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
